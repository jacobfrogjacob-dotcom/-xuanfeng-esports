# 玄封電競 - 網站規格文檔

## 1. 專案概述
- **名稱**: 玄封電競
- **類型**: 靜態網站 (單頁 HTML + LocalStorage)
- **用途**: 提供打手媒合服務，讓用戶可透過連結聯繫打手
- **技術**: 純 HTML + CSS + JS，免費託管

## 2. 頁面結構

### 2.1 用戶頁面 (index.html)
- 標題: 玄封電競
- 社群按鈕: YouTube、TikTok、Line、Discord
- 打手列表區塊

### 2.2 管理員頁面 (admin.html)
- 密碼驗證
- 新增/編輯/刪除打手
- 調整社群連結

## 3. 功能規格

### 3.1 用戶頁面
- 顯示社群按鈕 (YouTube/TikTok/Line/Discord)
- 顯示打手列表 (名稱、價格、遊戲、連結)
- 點擊連結可外部開啟

### 3.2 管理員系統
- 密碼登入 (預設: admin123)
- 新增打手 (名稱、價格、遊戲、連結)
- 編輯打手
- 刪除打手
- 設定社群連結

### 3.3 資料儲存
- 使用 LocalStorage 儲存所有資料
- 資料結構:
  ```json
  {
    "links": {
      "youtube": "",
      "tiktok": "",
      "line": "",
      "discord": ""
    },
    "boosters": [
      {
        "id": 1,
        "name": "打手名稱",
        "price": "價格",
        "game": "遊戲",
        "link": "連結"
      }
    ]
  }
  ```

## 4. UI/UX 規格

### 4.1 色彩系統
- 主色: #1a1a2e (深藍黑)
- 次色: #16213e (深藍)
- 強調: #e94560 (紅色)
- 文字: #ffffff (白色)
- 背景: #0f0f23 (極深藍)

### 4.2 字體
- 標題: "Noto Sans TC", sans-serif
- 內文: "Noto Sans TC", sans-serif

### 4.3 動效
- 按鈕 hover: scale(1.05) + 發光效果
- 卡片 hover: 浮起效果
- 淡入動畫
